Travelling around might confuse you in the end of the year if you want to see where you have been. This webapp allows you to track where you wake up. For me it is an example which allows me to test-drive the following HTML5-technologies

  • yeoman
  • Angular JS
  • Geolocate the user via webbrowser
  • HTML5 offline storage mechanism
  • D3 library for simple visualizing the data

Have fun using the app.
